## üõ†Ô∏è Technical Features

### AI Pipeline
- **Stage 1**: Multi-view synthesis with Zero123++ (6-8 consistent views)
- **Stage 2**: 3D reconstruction with TripoSR (512 resolution)
- **Stage 3**: Mesh cleanup (watertight + quad retopology)
- **Stage 4**: PBR texture generation (4K with ORM packing)
- **Stage 5**: Auto-rigging for characters (30 bones optimized)

### Supported Formats
- **Import**: Text prompts, images (PNG/JPG), videos (MP4), multi-view images
- **Export**: GLB, FBX, OBJ, USD + engine-specific metadata

### System Requirements
**Minimum**:
- CPU: 4 cores
- RAM: 8GB
- GPU: Integrated graphics (AMD/Intel)
- Storage: 15GB (10GB for models)

**Recommended**:
- CPU: 6+ cores
- RAM: 16GB
- GPU: GTX 1060 / RX 580 or better
- Storage: 25GB SSD

## üì¶ Quick Start

### Step 1: Install Dependencies
```bash
# Clone repository
git clone https://github.com/khvsreddy-code/MINEDEV.git
cd MINEDEV

# Install frontend
npm install

# Install backend
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
```

### Step 2: Download AI Models (One-Time, ~5-8GB)
```bash
python download_models.py
```

### Step 3: Start Services
```bash
# Terminal 1: Start backend
cd backend
python server.py

# Terminal 2: Start frontend
cd ..
npm run dev
```

### Step 4: Generate Your First 3D Model!
1. Open http://localhost:5173
2. Enter prompt: "a futuristic robot character"
3. Click "Generate 3D Model"
4. Wait ~2 minutes
5. Rotate/zoom the model
6. Export as GLB!
